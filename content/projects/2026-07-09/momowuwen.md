墨墨梧文是基于 Next.js 15 + React 19 构建的中国古典水墨风格个人博客系统。

## 技术栈

- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS v4 + 自定义水墨主题
- **内容管道**: Velite (Markdown → 结构化数据)
- **后台管理**: Decap CMS (GitHub-backed)
- **部署**: Vercel

## 特色

- 宣纸底色 + 朱红印章 + 水墨风格视觉
- CSS 液态玻璃导航栏
- 响应式布局支持移动端
- RSS / Open Graph / SEO 元数据
- 文章阅读量统计
